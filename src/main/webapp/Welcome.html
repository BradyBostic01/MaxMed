MaxMed

Large and reliable drug database

At the core MaxMed draws from the reliable and up to date DrugBank database. Search from Generic Name, Brand Name, DrugBank ID, or UNII (Unique Ingredient Identifier)

Drug Interactions

Search one substance or compare two

Access 2.4 Million known interactions with a single click, or find the interaction between two drugs in the same search bar

Things that made this app possible:

  • Java - backend api
  • - database management system implementing MySQL
  • DBParser - XML data parser used to parse the drug data
  • Bootstrap 4 - toolkit that made this app look nice (With HTML and a little CSS)
  • AngularJS - Javascript framework that made this app interactive
  • AWS - Amazon web service for email capabilities
  • Coffee - potent caffeine source
